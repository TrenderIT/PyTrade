# PyTrade - набор скриптов для автоматизации торговли биржевыми инструментами.

## Скрипты для ребалансировки портфеля

Скрипт при запуске вычисляет необходимые изменения на текущем счете, чтобы соотношение инструментов в рублевом выражении соответствовало заданным настройкам.

gui\robot_gui.py - скрипт для редактирования настроек портфеля
pytrader\Tinkoff Robot - readonly version.py - скрипт производящий ребалансировку

### Редактирование настроек (gui\robot_gui.py)
Скрипту в 1-м параметре командной строки передается имя файл с параметрами для правки.
Если параметр не задан, то используется имя скрипта, в котором раширение заменено на json.

В настройках задается:
- идентификатор счета для проведения балансировки
- доля баланса счета, не участвующая в балансировке
- инструменты (по figi) и их доля в общем объеме

FG0000000000 - figi используется для задания инструмента "рубль". Использование не обязательно, т.к. можно использовать параметр, задающий неиспользуемую долю счета.

### Ребалансировка портфеля (pytrader\Tinkoff Robot)
Скрипту в 1-м параметре командной строки передается имя файл с параметрами для запуска

Скрипт вычисляет необходимые изменения и выставляет ордеры.

Текущая версия работает в режиме "только для чтения"
